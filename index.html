neptune

Windows desktop app

유튜브 타입 비트의 BPM, Key, 추출 오디오까지

링크를 붙여넣으면 제목, 설명, 댓글, 오디오를 함께 분석해 BPM과 Key를 확인하고 권한이 있는 영상의 오디오 파일을 내려받을 수 있습니다.

최신 설치 파일: neptune.Setup.0.2.0-beta.1.exe · 약 247MB · Windows 10/11

Local first

각 사용자의 컴퓨터에서 처리

서버에서 YouTube 오디오를 대신 받는 방식이 아니라, 설치된 앱이 사용자 PC에서 변환과 분석을 실행합니다.

Included tools

별도 Python 설치 없이 실행

분석 런타임, yt-dlp, ffmpeg 경로 확인을 앱 안에서 처리하도록 구성했습니다. 사용자는 설치 후 바로 실행하는 흐름을 목표로 합니다.

Creator workflow

BPM과 Key 중심의 결과

타입 비트를 찾는 래퍼, 프로듀서, 송라이터가 바로 활용할 수 있도록 추천 BPM, 추천 Key, Camelot 정보를 간결하게 보여줍니다.

Install

설치 방법

  1. 위 다운로드 버튼을 눌러 setup.exe를 받습니다.
  2. Windows SmartScreen 경고가 뜨면 게시자 정보를 확인하고 실행합니다.
  3. 설치 후 neptune을 열고 YouTube 링크를 붙여넣습니다.
  4. 권리 확인 체크 후 분석 및 변환을 시작합니다.
